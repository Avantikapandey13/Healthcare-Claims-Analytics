# Healthcare-Claims-Analytics
End-to-end Healthcare Claims Analytics project using Python, Pandas, Power BI and DAX.
## Project Overview

This project analyzes healthcare insurance claims using Python and Power BI.

The objective is to identify claim trends, provider performance, patient demographics, and operational insights through interactive dashboards.

## Technologies Used

- Python
- Pandas
- NumPy
- Power BI
- DAX

## Dataset

Enhanced Health Insurance Claims Dataset

Records: 4,500+

Features:
- Claims
- Providers
- Procedure Codes
- Patient Demographics
- Claim Status
- Claim Amounts


## Dashboard Pages

### 1. Executive Summary

KPIs:
- Total Claims
- Total Claim Amount
- Average Claim Amount
- Approval Rate

### 2. Provider Analysis

- Cost by Specialty
- Claims by Specialty
- Location Analysis

### 3. Patient Demographics

- Age Group Analysis
- Gender Analysis
- Income Analysis
- Employment Analysis

### 4. Claims Operations & Trends

- Monthly Trends
- Claim Status Trends
- Submission Method Analysis
